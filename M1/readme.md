## Controles

| Tecla         | Função                                      |
|---------------|---------------------------------------------|
| X             | Rotacionar continuamente em torno do eixo X |
| Y             | Rotacionar continuamente em torno do eixo Y |
| Z             | Rotacionar continuamente em torno do eixo Z |
| R             | Parar a rotação                             |
| ESC           | Fechar o programa                           |
